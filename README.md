# 모두가 즐거운 라온 퀴즈대회 (Raon Quiz App)

이 프로그램은 팀 대항 퀴즈 대회를 진행하기 위한 웹 애플리케이션입니다.

## 🚀 시작하기

1.  **설치**: 터미널에서 `npm install`을 실행하여 필요한 패키지를 설치합니다.
2.  **실행**: `npm run dev`를 실행하면 로컬 서버가 시작됩니다 (보통 `http://localhost:5173`).

## 🎮 게임 진행 방법

### 1. 팀 설정 (Team Setup)
- 게임을 시작하면 팀 이름을 입력하는 화면이 나옵니다.
- 기본적으로 2팀이 설정되어 있으며, `+ 팀 추가` 버튼으로 최대 4팀까지 추가할 수 있습니다.
- 팀 이름을 모두 입력한 후 `게임 시작` 버튼을 누르세요.

### 2. 메인 화면 (Game Board)
- 5개의 카테고리(성경, 부천, 음식, 넌센스, 나라)와 100~500점의 문제판이 보입니다.
- 원하는 점수를 클릭하면 문제가 팝업으로 나타납니다.

### 3. 문제 진행 (Question Modal)
- **문제 확인**: 팝업에 문제가 크게 표시됩니다.
- **정답 확인**: `정답 확인` 버튼을 누르면 정답이 나타납니다.
- **점수 부여**: 정답을 맞춘 팀의 버튼을 클릭하면 해당 팀에게 점수가 부여되고 창이 닫힙니다.
- **점수 없음**: 아무도 맞추지 못한 경우 `닫기 (점수 없음)`을 눌러 창을 닫습니다.
- *참고*: 이미 선택한 문제는 회색으로 변하고 다시 선택할 수 없습니다.

### 4. 점수판 (Score Board)
- 화면 하단에 각 팀의 현재 점수가 표시됩니다.
- 점수 옆의 `+100`, `-100` 버튼을 사용하여 수동으로 점수를 조절할 수 있습니다.

## 🛠 관리자 및 호스트 기능 (Advanced Features)

화면 우측 상단에 숨겨진 컨트롤 버튼들이 있습니다.

### 👑 호스트 모드 (Host Mode)
- **Host OFF / ON 버튼**: 클릭하여 호스트 모드를 켤 수 있습니다.
- **기능**: 호스트 모드가 켜진 상태에서는 문제를 클릭했을 때, 정답을 공개하기 전에 **화면 왼쪽 상단에 작은 노란색 박스로 정답을 미리 볼 수 있습니다.** (진행자가 답을 미리 확인하고 유도할 때 유용합니다)

### ⚙️ 관리자 모드 (Admin Mode)
- **Admin 버튼**: 클릭하면 문제와 카테고리를 수정할 수 있는 관리자 패널로 이동합니다.
- **카테고리 수정**: 카테고리 이름 옆의 화살표(▶)를 누르면 펼쳐지며, 이름을 직접 수정할 수 있습니다.
- **문제 수정**: 각 점수별 문제와 정답을 자유롭게 수정할 수 있습니다.
- **미디어 업로드**:
    - **미디어 타입**을 `이미지`나 `오디오`로 변경하면 파일을 업로드할 수 있습니다.
    - 업로드된 이미지는 문제 화면에 함께 표시됩니다.
    - *주의*: 파일은 브라우저 저장소(localStorage)에 저장되므로, 너무 큰 파일(500KB 이상)은 피해주세요.
- **저장**: `변경사항 저장` 버튼을 눌러야 수정된 내용이 반영됩니다.
- **초기화**: `초기화` 버튼을 누르면 처음 상태(기본 문제)로 돌아갑니다.
